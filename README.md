# AI 口算练习

由 AI 生成自定义口算题的练习工具。打开即用，无需注册，数据存本地。

## 功能

- **AI 出题** — 调用 DeepSeek API 生成符合约束的口算题
- **详细解答** — 每道题附带 AI 给出的分步解题过程
- **答案核验** — 提交答案后自动判断对错，服务器模式使用 Python 精确计算
- **历史记录** — 保存所有练习记录，支持按今天/全部筛选
- **本地存储** — 数据保存在浏览器或本地，不经过任何第三方服务器
- **导入导出** — 支持练习记录备份和恢复

## 题目约束

| 项目 | 约束 |
|------|------|
| 数字 | 每个数字最多 2 位（如 5, 63, 13.2, 32/3） |
| 运算 | 最多 2 个加法、2 个减法、2 个乘法、2 个除法 |
| 结果 | -1000 ~ 1000 之间 |
| 小数 | 最多 3 位小数 |
| 分数 | 分子分母各最多 3 位 |

## 使用方法

### 方法一：双击启动（推荐）

1. 确保电脑已安装 [Python](https://www.python.org/downloads/)
2. 双击 **`启动.bat`**
3. 浏览器自动打开
4. 在「设置」页配置 DeepSeek API Key
5. 返回「练习」页，点击「生成题目」开始练习

### 方法二：直接打开浏览器

1. 双击 **`index.html`**
2. 在「设置」页配置 DeepSeek API Key
3. 返回「练习」页，点击「生成题目」开始练习

## 技术说明

- 纯前端：HTML + CSS + JavaScript（单个 `index.html` 文件）
- 后端可选：`server.py` 提供 HTTP 服务 + AI API 代理 + 答案精确核验
- AI 模型：[DeepSeek Chat](https://platform.deepseek.com/)
- 无需安装任何依赖包（纯 Python 标准库）

## 配置 API Key

1. 前往 [DeepSeek 平台](https://platform.deepseek.com/) 注册并获取 API Key
2. 在应用「设置」页输入 API Key 并保存
3. API Key 仅保存在你的浏览器本地，不会上传到其他服务器

## 输入答案格式

| 答案类型 | 示例输入 |
|---------|---------|
| 整数 | `68` |
| 小数 | `23.456` |
| 分数 | `697/63` |
| 负数 | `-68` |
